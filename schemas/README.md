
# `schemas` folder

- `common` - this folder contains all common UBL 2.4 definitions of elements and structures.
- `maindoc` - this folder contains the UBL documents that are used for ESPD. `UBL-QualificationApplicationRequest-2.4.xsd` is the basis for ESPD Request document, and `UBL-QualificationApplicationResponse-2.4.xsd` is the basis for ESPD Response document.
